# Простановка тэгов по языкам

*Языки: [English](index.md) · **Русский** — Родитель: [индекс репозитория](../index.md)*

Правила для конкретных языков — куда ставить токены `@tag:<slug>` **в коде**. В
любом языке тэг может маркировать четыре уровня — **пакет**, **файл**, **класс**
(или его аналог) и **метод/функцию** — и всегда записывается токеном
`@tag:<slug>` внутри комментария в синтаксисе языка, поставленного прямо над
маркируемым элементом.

Про общий механизм и про тэги в **документации** (YAML front matter, `index.md`,
тэги в теле) см. [`INSTRUCTIONS.ru.md`](../INSTRUCTIONS.ru.md) /
[`INSTRUCTIONS.md`](../INSTRUCTIONS.md) и [`README.ru.md`](../README.ru.md).

## Файлы

- [`java.md`](java.md) — тэги в Java (`//`, `/* */`, Javadoc; `package-info.java`).
- [`python.md`](python.md) — тэги в Python (`#`-комментарии и docstring; `__init__.py`).
- [`golang.md`](golang.md) — тэги в Go (`//`-doc-комментарии; `doc.go`).

Для языка, которого нет в списке, применяй ту же идею: токен `@tag:<slug>` в
комментарии (в синтаксисе этого языка) прямо над пакетом, файлом, аналогом класса
или функцией.
